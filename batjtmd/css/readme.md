## shirnk 收缩计算
核心：flex-shrink 缩小的时候不仅要考虑其权重还要考虑flex-basic来算总权重（flex-shrink*flex-basic），但是flex-grow则不用考虑flex-basic
- 溢出空间 300px 如何让子元素shrink 分摊
- 分摊的比例？
  flex-shrink: 2:1  500:400
  500 - 300*500* 需要分摊的比例
  500 - 300*500* 需要分摊的比例
- 分摊的权重
  总的权重？
  500*2 + 400*1 = 1400
  1000/1400 = 0.714 71.4%
  400/1400 = 0.286 28.6%
  500 - 300*0.714 = 286
  400 - 300*0.286 = 318.6
